🌌 3D Portfolio Website

A modern, interactive 3D developer portfolio built to showcase projects, skills, and experience using immersive web technologies.

📖 Overview

This project is a 3D web portfolio designed to stand out from traditional websites by using interactive visuals and animations.

Instead of static sections, the site leverages 3D rendering and motion effects to create a visually engaging user experience that highlights your work in a unique way.

✨ Features
🎯 Interactive 3D Experience
Built using modern 3D web technologies for immersive UI
⚡ Smooth Animations
Scroll-based and motion-driven transitions
📱 Responsive Design
Works across desktop, tablet, and mobile devices
📂 Project Showcase
Dedicated sections for projects, skills, and experience
📧 Contact Integration
Easy way for recruiters or collaborators to reach out
🛠️ Tech Stack
Frontend: React.js
3D Rendering: Three.js / React Three Fiber
Styling: Tailwind CSS / CSS
Animations: Framer Motion
Build Tool: Vite
⚙️ How It Works
3D models and scenes are rendered using Three.js
React manages UI components and state
Animations are triggered via scroll and user interaction
Sections (Hero, About, Projects, Contact) are structured as modular components
🚀 Getting Started

Clone and run locally:

git clone https://github.com/san-codes9234/three_d_portfolio.git
cd three_d_portfolio
npm install
npm run dev

Then open:

http://localhost:5173
